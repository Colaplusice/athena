# 基于卷积神经网络的人脸识别系统

## feature

1. 收集多个用户的照片，上传一个含有多个头像的照片，识别出每个人。
2. 拍照或者上传照片文件，进行分析结果，对年龄和性别进行识别，标注生成图片，返回结果
3. 用户登录和注册功能，用户上传图片，添加自己照片到资料库中，
添加到资料库中有两种办法: 1. 上传一张自己的照片。2. 客户端拍照添加。添加完毕的照片会显示到个人信息页面



## 运行

docker-compose up
